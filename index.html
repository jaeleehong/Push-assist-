🎴 뉴맞고_ 시즌레이스 푸시 메시지 자동 생성 도구

기획서를 복사 붙여넣기하면 푸시 메시지 발송일과 내용이 자동으로 생성됩니다

📋 사용방법

  1. 주제어 입력: 해당 월의 시즌레이스 주제를 입력하세요 (예: 더위사냥, 로맨틱, 추석맞고 등)
  2. 기획서 복사: 엑셀 기획서에서 날짜가 포함된 부분을 복사하세요
  3. 붙여넣기: 복사한 내용을 텍스트 영역에 붙여넣으세요
  4. 생성: "푸시 메시지 생성" 버튼을 클릭하세요

🎯 자동 생성되는 메시지

  • 시작 푸시: 목요일 오후 6시 (전체 대상)
  • 순위 하락 알림: 금,토,일 오후 1시, 7시 (정보분석대상자)
  • 피크타임: 토,일 오후 8시 (정보분석대상자)
  • 마감 임박: 일 오후 6시 (정보분석대상자)
  • 보상 지급: 월요일 오후 5시 (정보분석대상자)
